## ğŸ“– Description

The **Social Media Campaign Generator** is a state-of-the-art AI-powered application designed to streamline the content creation process for digital marketers, influencers, and businesses. Powered by **Doodax** and the **Google Gemini API**, this tool takes simple product descriptions and brand voice directives to generate tailored, viral-ready content across multiple platforms instantly.

This project represents the future of automated marketing, providing a seamless bridge between raw ideas and polished, platform-specific copy.

### Key Features
- **Multi-Platform Support:** Generates optimized content for Twitter, Instagram, LinkedIn, Facebook, and Quora simultaneously.
- **AI-Powered Creativity:** Uses advanced LLMs to match specific brand voices (witty, professional, futuristic, etc.).
- **Image Ideation:** Provides detailed prompts for designers to create matching visuals for Instagram.
- **SEO Optimized:** Built with JSON-LD Schema and semantic HTML for maximum search engine visibility.
- **Galaxy UI:** Features a stunning, animated cosmic interface for an immersive user experience.
- **Privacy First:** No data storage; all processing is ephemeral.
